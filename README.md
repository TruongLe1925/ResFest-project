# University Admission Prediction

University admission prediction system - React + Spring Boot + AI

## Run Frontend

```bash
cd admissionPredic-main
npm install
npm run dev
```

- URL: http://localhost:5173

## Run Backend (Docker Compose)

```bash
cd universityDocker
docker-compose up -d
```

- API: http://localhost:8080
- Swagger: http://localhost:8080/my-ui.html

## Stop Backend

```bash
cd universityDocker
docker-compose down
```
